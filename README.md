# Customer Service Chatbot

## 📌 Project Description

A rule-based customer service chatbot developed using Python and JSON. The chatbot can answer customer queries related to orders, refunds, shipping, payments, account issues, and support.

## 🚀 Features

* Greeting responses
* Order tracking queries
* Refund and return information
* Shipping details
* Payment methods
* Contact support
* Chat history logging
* Handles 300+ query variations

## 🛠️ Technologies Used

* Python
* JSON
* VS Code

## ▶️ How to Run

```bash
python chatbot.py
```

## 📷 Sample Queries

* Hello
* Where is my order?
* I want a refund.
* How can I pay?
* Contact customer support.
* Forgot my password.
