# Customer Service Chatbot

## Description
A rule-based chatbot developed using Python for answering customer queries related to orders, refunds, payments, shipping and support.

## Features
- Greeting responses
- Order status inquiry
- Refund information
- Payment methods
- Shipping details
- Order cancellation
- Contact support
- Chat history logging

## Technologies Used
- Python
- JSON

## How to Run

python chatbot.py

## Future Scope
- Add AI and NLP capabilities
- Integrate with websites
- Add database support
